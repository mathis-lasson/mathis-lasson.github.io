GateKeepr

GateKeepr is a USB threat detection station designed to identify and eliminate common threats present on any USB device (keyboard, mouse, storage, USB key). Inspired by white stations, it also includes a keylogger that analyzes all keys and commands to protect the user from 'Bad USBs' such as Rubber Ducky or Bash Bunny. For all storage devices, it uses the ClamAV open source project to scan for malicious files.
